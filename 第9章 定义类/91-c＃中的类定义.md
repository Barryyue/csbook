##C#中的类定义

&emsp;&emsp;C#使用 `class` 关键字来定义类：

```javascript
        class MyClass
        {
            // Class members.
        }
```

&emsp;&emsp;这段代码定义了一个类 `MyClass`。定义了一个类后，就可以在项目中能访问该定义的其他位置对该类进行实例化。默认情况下，类声明为内部的，即只有当前项目中的代码才能访问它。可使用 `internal` 访问修饰符关键字来显式地指定这一点，如下所示（但这是不必要的）：

```javascript
        internal class MyClass
        {
            // Class members.
        }
```
