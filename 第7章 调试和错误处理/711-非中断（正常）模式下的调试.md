##非中断（正常）模式下的调试

&emsp;&emsp;本书经常使用的一个命令是 `Console.WriteLine()` 函数，它可以把文本输出到控制台。在开发应用程序时，这个函数可以方便地获得操作的额外反馈，例如：

```javascript
        Console.WriteLine("MyFunc() Function about to be called");
        MyFunc("Do something.");
        Console.WriteLine("MyFunc() Function execution completed.");
```
