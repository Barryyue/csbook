（1）下面两个函数都存在错误，请指出这些错误。

```javascript
        static bool Write()
        {
            Console.WriteLine("Text output from function.");
        }

        static void MyFunction(string label, params int[] args, bool showLabel)
        {
            if (showLabel)
                Console.WriteLine(label);
            foreach (int i in args)
                Console.WriteLine("{0}", i);
        }
```
