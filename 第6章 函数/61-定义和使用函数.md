##定义和使用函数

&emsp;&emsp;本节介绍如何将函数添加到应用程序中，以及如何在代码中使用（调用）它们。首先从基础知识开始，看看不与调用代码交换任何数据的简单函数，然后介绍更高级的函数用法。首先分析一个示例：

>把下述代码添加到 `Program.cs` 中：

>```javascript
        class Program
        {
            static void Write()
            {
                Console.WriteLine("Text output from function.");
            }
            static void Main(string[] args)
            {
                Write();
                Console.ReadKey();
            }
        }
```
